Welcome To Website Development 101 Course

Website development (also called web development) is the process of creating and maintaining websites.
It involves all the work that goes into building a website, from the structure and design to how it functions and looks on the internet.

Website Develpopment Divided in to Three groups

Frontend Development | Backend Development| Fullstack Development

Frontend Development

Frontend development is the part of web development that focuses on the visual side of a website — what users see and interact with directly in their browser.

Backend Development

Backend development is the part of web development that works behind the scenes —
it’s everything that happens on the server side to make a website function properly. Users don’t see this part directly, but it powers the features they interact with on the frontend.

Fullstack Development

Full-stack development means working on both the frontend and the backend of a website or web application. A full-stack developer is someone who can build the entire system —
from what the user sees in the browser to how the server and database handle data in the background.
